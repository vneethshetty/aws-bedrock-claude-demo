\# AWS Bedrock Claude Integration (Python)



This project demonstrates how to integrate Amazon Bedrock foundation models using Python (boto3).



\## Features

\- Uses AWS IAM authentication

\- Invokes Anthropic Claude via inference profile

\- Accepts dynamic user input

\- Returns AI-generated response



\## Tech Stack

\- AWS Bedrock

\- Anthropic Claude 3 Sonnet

\- Python

\- Boto3



\## Setup Instructions



1\. Install Python

2\. Create virtual environment:

&nbsp;  python -m venv venv

&nbsp;  venv\\Scripts\\activate



3\. Install dependencies:

&nbsp;  pip install boto3



4\. Configure AWS CLI:

&nbsp;  aws configure



5\. Run:

&nbsp;  python app.py



\## Use Case



Demonstrates practical AI integration within cloud-native DevOps environments.
